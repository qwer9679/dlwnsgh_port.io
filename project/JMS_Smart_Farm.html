프로젝트 개요

JMS Smart Farm은 Arduino와 Python을 활용하여 스마트팜 시스템을 구축한 프로젝트입니다.
이 시스템은 온도, 습도, 조도 센서를 통해 환경 데이터를 수집하고, 이를 기반으로 자동으로
물주기 및 조명 제어를 수행합니다.
또한, 웹 인터페이스를 통해 실시간 모니터링과 제어가 가능합니다.

👊 사용 기술

🌱 Arduino
Arduino 센서를 통해 온도, 습도를 측정하여 전달, 제어하는데 사용
🐍 python
Arduino에서 전달한 데이터를 DB, 프론트엔드에 전달하는데 사용
🐬 MySQL
온도, 습도, LED, 팬, 펌프 등의 데이터를 저장하는데 사용
🐳 Docker
컨테이너 기반의 배포 및 환경 관리에 사용
☁️ Google Cloud Platform
Google Drive를 통한 유튜브 라이브 자동 생성에 사용
